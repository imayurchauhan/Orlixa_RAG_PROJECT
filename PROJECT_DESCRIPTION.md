# Orlixa — Intelligent Hybrid RAG + Web Search Assistant

## 1. Project Summary
**Orlixa** is a state-of-the-art, full-stack AI conversational assistant designed to bridge the gap between static internal knowledge and the dynamic, real-time nature of the internet. By integrating **Retrieval-Augmented Generation (RAG)** with a high-performance **Web Search Engine** and an intelligent **Query Routing Layer**, Orlixa provides users with highly accurate, context-aware, and grounded responses. Whether the query involves deep technical documentation stored in a private knowledge base or the latest live news from the web, Orlixa intelligently synthesizes information to deliver the most relevant answers.

---

## 2. Full Technical Description
Orlixa operates on a multi-stage pipeline designed for precision and speed. At its core, the system utilizes an **Intent-Based Router** that classifies incoming queries to determine the optimal retrieval path: **RAG**, **Live Web Search**, or a **Hybrid** combination of both.

*   **RAG Engine:** Leverages high-dimensional vector embeddings and semantic search to retrieve relevant context from indexed documents. This ensures the assistant's answers are grounded in specific, trusted data, making it ideal for technical support and enterprise knowledge retrieval.
*   **Web Search & Scraping Engine:** For real-time queries (e.g., "What is the gold price in Surat today?" or "Latest IPL scores"), the system triggers a web search pipeline. It fetches results using high-concurrency APIs, scrapes content from top-ranked pages, and employs an advanced **Content Extraction & Cleaning** module to strip away HTML noise (ads, scripts, navigation) and extract raw, readable text.
*   **Information Fusion:** The final response is generated by a Large Language Model (LLM) that performs **Source-Aware Reasoning**. It fuses data from multiple sources, prioritizes trusted domains, and cites its sources to eliminate hallucinations and ensure transparency.

---

## 3. Feature List
*   **Intelligent Intent Routing:** Automatically detects if a query is technical, real-time, location-based, or hybrid.
*   **Hybrid Retrieval:** Merges internal document knowledge with real-time web data for comprehensive answers.
*   **Real-Time Data Handling:** Supports live updates for news, weather, sports scores, and financial markets.
*   **Advanced Web Scraping:** Parallelized fetching and intelligent HTML cleaning for fast, high-quality data extraction.
*   **India-Specific Intelligence:** Optimized for location-aware queries (e.g., city-specific prices and local news).
*   **Source Verification:** Prioritizes trusted domains and technical documentation, providing direct links to sources.
*   **Low-Hallucination Design:** Strict grounded-response logic ensures the AI only speaks from provided or retrieved context.
*   **Responsive UI:** A modern, dark-themed React/Next.js chat interface with smooth micro-animations.

## 4. Architecture Overview

### **Frontend**
*   **Framework:** Next.js (React)
*   **UI/UX:** Vanilla CSS with a focus on premium, dynamic design (glassmorphism, vibrant gradients).
*   **Features:** Real-time message streaming, responsive layout for mobile/desktop, and session-based chat history.

### **Backend**
*   **Framework:** Python (FastAPI)
*   **Query Router:** A logical layer that classifies queries using LLM-based intent detection.
*   **Execution Engines:** Dedicated modules for RAG (semantic search) and Web Search (scraping & cleaning).
*   **LLM Integration:** High-speed inference using Groq (Llama-3/Mixtral models).

### **Data Layer**
*   **Vector Database:** Used for storing and retrieving document embeddings for RAG.
*   **Relational Database:** SQLite for user management, chat history persistence, and persona templates.

### **External Services**
*   **Search APIs:** Serper / Tavily for high-quality web result fetching.
*   **Inference:** Groq Cloud for ultra-fast response generation.

## 5. Use Cases

## 6. Benefits
*   **Accuracy & Trust:** By citing sources and prioritizing verified domains, Orlixa builds user confidence.
*   **Extreme Speed:** The combination of FastAPI and Groq ensures near-instant response times.
*   **Always Up-to-Date:** Unlike static LLMs, Orlixa’s web search ensures its knowledge never expires.
*   **Scalability:** The modular architecture allows for easy addition of new search tools or vector stores.
*   **Privacy-First:** Designed with user-level isolation, ensuring personal data and chat history remain secure.
